Detalhes do Projeto

Este website foi criado no âmbito do projeto de apetidão profissional da secundária, foram utilizados vários templates bootstrap para chegar ao resultado final, e foram ainda utilizadas as tecnologias de Javascript para ir buscar dados ao front-end e de seguida através de Ajax enviar para o back-end, onde são feitas todas as verificações possíveis para efetuar as operações CRUD desejadas.

Este tem como objetivo os fãs da marca UMM possam recorrer para efetuar compras de peças para todos os veiculos, o website contém ainda um Espaço do Utilizador onde os clientes do mesmo poderão adicionar, alterar a quantidade e remover produtos do carrinho de compras e finalizar as suas encomendas.
