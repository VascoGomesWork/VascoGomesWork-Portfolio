Detalhes do Projeto

O projeto da unidade curricular de Programação de Dispositivos Móveis foi realizado em conjunto com o meu colega Fernando Simões. Este tem como objetivo realizar um classificador de insetos, em que para isso foi necessário ter acesso à camera do telemóvel, assim como ter acesso ao gestor de ficheiros do dispositivo em questão e ainda ser capaz de captar a sua posição de GPS no momento da fotografia caso o utilizador deixe.

Neste projeto foi utilizado a estrutura MVVM - Model View, View Model para conseguir estruturar o projeto. Foram utilizadas Recycler views, que consistem em ter uma lista com elementos recicláveis, poupando assim memória do dispositivo, uma vez que quando um elemento da lista sai da atividade, este é reciclado para conter outro conteúdo, fazendo assim que apenas seja gerada uma lista com os elementos necessários.
