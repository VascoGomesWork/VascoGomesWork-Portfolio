Detalhes do Projeto

O projeto apresentado nesta página foi realizado no âmbito do meu estágio curricular da Licenciatura de Engenharia Informática, tendo este sido realizado na empresa RedShift III - Innovation com escritório em Beja, a empresa possui um ambiente de trabalho bastante descontraido, tirando ainda todas as dúvidas assim como ajudaram-me no crecimento pessoal enquanto desenvolvedor web a trabalhar com a framework Angular, providenciando todos os cursos necessários para aprender acerca das tecnologias utilizadas, como por exemplo o RxJS e o NgRx.

O RxJS é uma biblioteca que permite o uso de Observables para facilitar pedidos assíncronos.

Já o NgrX é uma framework que permite fazer uso do State, assim como de isolamento de Effects através de conexão de Observables de ações com a Store.
