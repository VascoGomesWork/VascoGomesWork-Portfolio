Detalhes do Projeto

Este projeto foi realizado no contexto da unidade curricular de Tecnologias Web e Ambientes Móveis, foi a primeira abordagem que tive com React, assim como desenvolvimento com bibliotecas de javascript para front-end.

Para a realização do projeto foi utilizada a biblioteca de javascript React, que em conjunto com a API Spoonacular me permitiu construir um sistema web em que os objetivos são mostrar vários planos alimentares vindos da API na página inicial, e de seguida realizar a criação de um plano alimentar personalizado com diversos tipos de alimentos e receitas disponibilizadas pela API da Spoonacular, e ainda conseguir visualizar os planos alimentares criados e os seus detalhes.

No meu GitHub, está presente o front-end deste projeto.

Uma vez que não tinha nenhuma experiência com React, segui este curso do Scrimba que me ajudou a aprender a trabalhar com React e de seguida a implementar neste projeto.
