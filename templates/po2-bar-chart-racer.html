Detalhes do Projeto

Este projeto foi realizado no âmbito da unidade curricular de Programação Orientada por Objetos, onde foi pedido que programasse um jogo do estilo Bar Chart Race onde é possível escolher o ano de um ficheiro CSV dado pelo professor da unidade curricular, e desta maneira apresentar o gráfico para esse ano.

É ainda possível animar os gráficos, através do uso de Threads em Java, fazendo a chamada corrida de barras passando por todos os anos, estando estes ordenados. Faz ainda parte do enunciado a necessidade de existirem 2 skins para os gráficos, sendo estas as barras com cores aleatórias e ainda as barras com linhas.

Para ganhar pontos extra, pedia-se ainda extras, sendo um dos meus o Dark Theme, uma vez que passei bastante tempo à noite a programar, senti a necessidade de existir um fundo branco para poupar os meus olhos. Realizei a mudança de cores das letras como outro extra do programa.

O repositório deste projeto está disponível no meu GitHub.
