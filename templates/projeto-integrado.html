Detalhes do Projeto

O projeto em questão foi escolhido por mim e pelo meu colega de curso, Fernando Simões na unidade curricular de Projeto Integrado.

Este consistiu no desenvolvimento de um sistema de gestão de stocks para o laboratório SEPSI, onde o professor orientador do projeto nos deixou à vontade em relação às tecnologias de desenvolvimento que podiamos utilizar, sendo que por esse mesmo motivo decidimos utilizar React, uma vez que já haviamos desenvolvido projetos com a biblioteca de javascript, e de ser simples de utilizar. Para o back-end foram utilizadas as tecnologias Flask esta é uma API feita em Python e SQLAlchemy que é a uma biblioteca que realiza mapeamentos entre objetos e relações, esta em questão foi desenvolvida para a linguagem de programação Python.

O projeto em questão tem várias funcionalidades, entre estas as requisições, devoluções e criações de kits de materiais. Os kits de materiais foi um dos requisitos pedidos pelo professor orientador, uma vez que pode ser necessário requisitar múltiplas vezes os mesmos materiais, e desta maneira ao invés de se realizar requisições material a material, são apenas requisitados multiplos kits de materiais.

As requisições podem ser efetuadas com múltiplos materiais, ferramentas entre outro tipo de equipamentos que pode ser introduzido no sistema, assim como kits de materiais e requisitar tudo de uma vez. Já as devoluções pode ser devolvido parte dos materiais, ferramentas, equipamentos e kits presentes numa requisição.
